# Bayesian Data Analysis - Project Portfolio

Welcome to my Bayesian Data Analysis repository! This portfolio documents my practical projects focusing on Bayesian statistical inference, Markov Chain Monte Carlo (MCMC) methods, and their applications to financial markets and economic data analysis.

## Technical Stack & Core Skills

* **Programming & Libraries**: Python (PyMC, ArviZ, Pandas, NumPy, Matplotlib)
* **Core Methodologies**: Bayesian Inference, Markov-Switching Models, Mixture Models
* **Algorithms & Implementation**: Gibbs Sampling, Metropolis-Hastings Algorithm, Posterior Distribution Estimation
* **Model Evaluation**: MCMC Chain Convergence Diagnostics, Trace Plots, Autocorrelation Analysis
